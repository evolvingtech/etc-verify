# ETC Verify Documentation

Documentation for the ETC Verify substrate.

## Structure

- `concepts/` — foundational concepts: the Contract type, side conditions, silences as typed data, the Shannon-style algebra/network correspondence.
- `tutorials/` — guided walkthroughs for new users.
- `api/` — auto-generated API reference (when `doc-gen4` is wired up).

Documentation lands as the corresponding substrate modules mature.
