# ETCVerify.Diagram

This subdirectory is reserved for the bidirectional algebra–diagram roundtrip
implementation described in `2026-05-16_vision-and-roadmap_etc-verify-framework.md`
§3.

No source files are present in this directory in v0.2.x. The bidirectional
roundtrip is planned for a later release. Substrate operators and types
defined in `ETCVerify/Core/` and `ETCVerify/Operators/` are designed to
preserve roundtrip-compatibility (see vision §9 and the May 19 handoff §4.5 —
"typeclasses for identity, structure fields for values").
